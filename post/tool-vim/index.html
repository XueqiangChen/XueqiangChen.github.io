Vim 折腾记

Mar 16, 2021 · 5 min read · vim  ·
分享到:

折腾了两天的vim，想要把它变成我的机器上默认的IDE，方便在没有环境的时候，快速查阅代码，做一些基本的开发任务，折腾过程记录在这篇博客上

环境

1# cat /etc/*release
2CentOS Linux release 7.9.2009 (Core)

系统自带的vim是7.x，对于很多的插件来说(例如 ycm)都已经不兼容了，所以第一步就是要将 vim 的版本更新到 8.0 以上

安装软件

编译vim之前先卸载掉旧版本的vim

1yum remove vim-enhanced vim-common vim-filesystem vim-minimal

为了编译能够正常，需要安装上一些依赖和常用的工具包：

1# install devtoolset-8
2yum install centos-release-scl
3yum-config-manager --enable rhel-server-rhscl-7-rpms
4yum install devtoolset-8
5scl enable devtoolset-8 bash
6
7# install cmake, python3-devel, etc.
8yum -y install git ncurses-devel ruby ruby-devel lua lua-devel perl perl-devel python3 python3-devel python2-devel perl-ExtUtils-Embed lrzsz cmake wget gcc gcc-c++ unzi

其中有部分工具是给后面安装插件使用的。

下载vim的源代码：

 1git clone https://github.com/vim/vim.git
 2cd vim
 3# if you build vim before
 4make distclean
 5# config
 6# --enable-fail-if-missing 表示问题会提示报错，并停止
 7# --enable-***interp=yes 表示加入***支持
 8# --with-***-config-dir=*** 表示指定配置文件路径
 9./configure --with-features=huge \
10            --enable-rubyinterp=yes \
11            --enable-luainterp=yes \
12            --enable-perlinterp=yes \
13            --enable-python3interp=yes \
14            --enable-pythoninterp=yes \
15            --with-python-config-dir=/usr/lib64/python2.7/config \
16            --with-python3-config-dir=/usr/lib64/python3.6/config-3.6m-x86_64-linux-gnu \
17            --enable-fontset=yes \
18            --enable-cscope=yes \
19            --enable-multibyte \
20            --disable-gui \
21            --enable-fail-if-missing \
22            --prefix=/usr/local \
23            --with-compiledby='Professional operations'
24# 编译
25# make VIMRUNTIMEDIR=*** 表示指定VIM可执行文件的位置
26make VIMRUNTIMEDIR=/usr/local/share/vim/vim82 && make install

编译完成之后，查看一下vim的版本：

1$ vim --version
2VIM - Vi IMproved 8.2 (2019 Dec 12, compiled Mar 16 2021 16:12:42)

配置与插件

为了支持go代码的开发，这里先以go的IDE环境为示例，

首先，插件的管理工具，这里选择 vim-plug。在Linux环境下直接用curl下载即可：

1curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
2    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

插件的配置也非常简单，只要将所有的插件配置在 call plug#begin('~/.vim/plugged') 和 call plug#end() 之间即可，常见的插件基本上都可以从 github 中找到，如果 github 找不到的话基本上 vim.org 的脚本都可以在 vim-script 中找到备份
